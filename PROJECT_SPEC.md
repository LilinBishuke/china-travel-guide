## プロジェクト概要

中国渡航を予定する外国人旅行者向けのコンパニオンアプリ。  
「ビザ判定→書類準備→出発前チェック→現地活用」を一本でカバーする。

**核心的価値：**  
中国固有の障壁（GFW・QR決済・ビザ複雑性）を知らない外国人旅行者の不安を、出発前から解消する信頼できるガイド。

**運用費：¥0/月**（Cloudflare Pages + 無料API群）

---

## ターゲットユーザー

| セグメント | 割合 | 主なニーズ |
|---|---|---|
| 初訪問観光客 | 50〜60% | ビザ・GFW・決済の不安解消 |
| リピーター | 20〜25% | 深い情報・旅程管理 |
| ビジネストラベラー | 15〜20% | ビザ確認・効率的な移動 |

**主ターゲット：初訪問観光客（欧州・日本・韓国・豪州・北米）**

## UXフロー

### オンボーディング（初回のみ・7ステップ）

```
[1. Welcome]          → 言語選択
[2. Passport]         → 国籍 + 旅行目的（観光/ビジネス/トランジット）
[3. Travel Style]     → 🚶徒歩 / 🚲サイクル / 🏍バイク / 🚗ドライブ / 🚌ツアー / 🚆鉄道
[4. Entry Details]    → 入国ポート + 出発日（リマインダー計算基準）
[5. Eligibility]      → ビザ判定（4分岐）
[6. Interests]        → 文化/食/自然/ショッピング/アドベンチャー
[7. Prompt]           → 「準備を始めましょう」→ PREPAREタブへ
```

**Eligibility 4分岐：**
- ✅ 30日ビザ免除（55カ国、2026年12月末まで）
- ✅ 240時間トランジット免除（経路条件明示）
- 📝 ビザ申請必要（進捗トラッカーへ誘導）
- 🌐 eVisa対応（COVAへリンク）

### メインアプリ（5タブ）

```
[PREPARE] [EXPLORE] [MY TRIP] [RATES] [TOOLS]
```

## 機能仕様

### PREPARE タブ

#### ビザ進捗トラッカー
ユーザーが申請の進捗を入力すると、次のステップへの準備状況を表示・リマインドする。

| Step | 内容 | ロック解除条件 |
|---|---|---|
| Step 1 | 書類チェックリスト | 全項目チェック完了 |
| Step 2 | COVA申請（申請番号入力） | 申請番号記録 |
| Step 3 | パスポート郵送（追跡番号入力） | 追跡番号記録 |
| Step 4 | 審査待ち（カウントダウン） | ユーザーが完了を報告 |
| Step 5 | ビザ確認（種別・期限・回数入力） | 全情報入力 |

**書類チェックリスト（全種共通）：**
- パスポート（有効期限6ヶ月以上）
- 証明写真（白背景・規定サイズ）
- 航空券予約（仮予約可）
- ホテル予約（仮予約可）

**追加書類（目的別）：**
- L（観光）：旅程表
- M（商用）：招待状・会社証明
- X（留学）：JW202フォーム
- F（交流）：招待状
- Q（家族）：家族関係証明書

#### 出発前タイムライン（リマインダー付き）

| タイミング | タスク |
|---|---|
| 出発100日前 | ビザ申請準備開始 |
| 出発90日前 | VPN契約・設定（中国入国後は設定不可） |
| 出発30日前 | WeChat登録（現地での登録は困難） |
| 出発14日前 | Alipay国際版設定・外国カードリンク |
| 出発7日前 | オフラインマップダウンロード |
| 出発前日 | eSIM or SIMフリー端末確認 |

#### GFWガイド

| ブロックされるアプリ | 代替手段 |
|---|---|
| Google Maps | Amap（高徳地図）|
| Gmail / Google系 | QQ Mail / WeChat |
| Instagram / X | VPN必要 |
| WhatsApp | WeChat |
| Uber | DiDi |
| YouTube | Bilibili |

#### 決済セットアップ
1. Alipay国際版（外国カード対応、16言語）
2. WeChat Pay（既存ユーザー認証が必要な点を説明）
3. BOC Compass（外国人専用・2025年新登場）

### EXPLORE タブ

- 都市別情報（オフライン対応）
- 地図（Leaflet + OpenStreetMap、オフラインキャッシュ）
- 天気予報（Open-Meteo API）
- 旅行スタイル別ガイド：

| スタイル | 主な情報 |
|---|---|
| 🚲 サイクリング | Hellobike/Meituan利用法・禁止エリア（杭州はガソリン原付禁止）|
| 🏍 バイク | 都市別禁止エリア（広州・深圳・北京）・外国人免許取得要件 |
| 🚗 ドライブ | 外国免許は無効・中国免許or現地ドライバー雇用・ナンバー規制 |
| 🚶 ハイキング | チベット入域許可証（TAP）・要ガイドエリア |
| 🚌 ツアー | ガイド必須エリア・日本語/英語ツアー手配 |
| 🚆 鉄道 | 高鉄予約方法（外国人向け）・Trip.com経由推奨 |

---

### MY TRIP タブ

- 旅程テンプレート（北京3日/上海5日/成都4日/西安3日/雲南5日…）
- AI旅程プランナー（入力フォーム→Claude.ai/ChatGPTへ自動プロンプト生成・リンク）
- 予約リンク（Trip.com 外国人可フィルター付き）

---

### RATES タブ

- CNY↔主要10通貨リアルタイム（Frankfurter API、ECBデータ、APIキー不要）
- 簡易電卓（金額入力→CNY換算）
- 最終更新時刻表示

---

### TOOLS タブ

- 翻訳（Google Translate ディープリンク）
- 緊急連絡先（国籍別大使館・救急110/119/120、オフライン）
- DiDi クイックランチャー
- Amap クイックランチャー

---

### PROFILE / SETTINGS（右上アイコンからアクセス）

アクセス方法：ダッシュボード右上のプロフィールアイコンをタップ

#### マイプロフィール
- ユーザー設定サマリー（国籍・旅行目的・旅行スタイル・出発日）
- 各設定の変更ボタン

#### ビザ・通知設定
- ビザポリシー更新通知（ON/OFF）
- リマインダー通知（ON/OFF）
- 通知タイミングの調整

#### 表示設定
- UI言語の変更（英語/日本語/韓国語/仏語/独語/西語）
- 通貨表示の基準（JPY/USD/EUR/GBP…）

#### オフラインデータ管理
- ダウンロード済みデータのサイズ表示
- キャッシュのクリア
- 手動でのオフラインデータ更新

#### アプリ情報
- バージョン表示
- プライバシーポリシー
- フィードバック送信リンク
- ビザ情報の免責事項

## アウトプット形式（検討中）

### 案A：PWA（推奨・フェーズ1）
- Cloudflare PagesでホスティングされたPWA
- URLシェアのみで配布、App Store不要
- オフライン・通知・インストール対応
- **維持費：¥0/月**

### 案B：静的Webサイト
- PWA機能なし（オフライン・通知なし）
- 最もシンプルな実装
- **維持費：¥0/月**

### 案C：PWA + App Store（フェーズ2）
- 案AをCapacitor.jsでラップしてネイティブアプリ化
- Apple Developer $99/年、Google Play $25一回
- **維持費：$99/年**

**推奨ロードマップ：案A→（ユーザー獲得後）案C**

## 競合との差別化

| アプリ | 強み | 弱点（このアプリが補う） |
|---|---|---|
| Trip.com | 予約最強 | 入国準備情報なし |
| Amap | ナビ最強 | ビザ・GFW情報なし |
| BOC Compass | 外国人専用 | 認知度低・決済特化 |
| Lonely Planet | 深いコンテンツ | 更新遅延・インタラクティブ性なし |
