# China Travel Guide PWA

外国人旅行者向けの中国渡航コンパニオンアプリ。ビザ判定からGFW対策、決済設定、観光地情報まで、中国旅行に必要なすべてをカバーするゼロコストPWA。

**Live**: https://china-travel-guide-ashen.vercel.app

---

## 概要

中国への渡航は、GFW（グレートファイアウォール）によるインターネット制限、独自の決済エコシステム（WeChat Pay / Alipay）、ビザ制度の複雑さなど、他国にない独特のハードルがあります。このアプリはそれらを一元的にガイドし、出発前の準備から現地での活用まで支援します。

### 対象ユーザー
- 中国に初めて渡航する外国人旅行者
- 観光・ビジネス・トランジット目的の渡航者
- 6言語対応（日本語・英語・韓国語・フランス語・ドイツ語・スペイン語）

### 技術スタック
- **フロントエンド**: React 18 + TypeScript + Tailwind CSS
- **ビルド**: Vite 5 + vite-plugin-pwa
- **ホスティング**: Vercel（ゼロコスト）
- **データ**: クライアントサイドのみ（サーバーなし）
- **永続化**: localStorage + IndexedDB（idb）
- **外部API**: Frankfurter（為替）、Open-Meteo（天気）、Wikivoyage（都市情報）

## 機能一覧

### オンボーディング（7ステップ）
| ステップ | 画面 | 内容 |
|---------|------|------|
| 1 | Welcome | アプリ紹介 + 言語選択（6言語） |
| 2 | Passport | 国籍選択 + 渡航目的（観光/ビジネス/トランジット） |
| 3 | Style | 旅行スタイル選択（徒歩/自転車/バイク/車/ツアー/高速鉄道） |
| 4 | Visa Result | ビザ判定結果の表示（免除/240h/eVisa/要申請） |
| 5 | Entry | 入国空港（10空港）+ 出発日 |
| 6 | Interests | 興味・関心の選択（文化/グルメ/自然/ショッピング/エンタメ/ウェルネス） |
| 7 | Complete | プロフィール確認 + 緊急タスク通知 |

### 準備タブ（メイン）
- **ダッシュボード**: 出発カウントダウン（大きな数字）、準備進捗バー、ビザ状況カード、次のタスクハイライト、為替レートウィジェット（LIVE）、クイックアクション（天気/地図/翻訳）、リマインダー
- **出発前タイムライン**: 90/60/30/14/7/1日前のマイルストーンとチェックリスト連動
- **チェックリスト**: 11項目（ビザタイプに応じて動的に変化）
- **ビザ情報**: 55カ国のビザ免除判定、ポリシー有効期限表示
- **ビザ進捗トラッカー**: 5ステップの申請進捗管理
- **GFWガイド**: ブロックサービス8件と代替アプリ、VPN比較4社、GFW対策チェックリスト（永続化）
- **決済設定**: WeChat Pay / Alipayのステップバイステップガイド
- **緊急連絡先**: 警察110/救急120/消防119 + 15カ国の大使館情報 + 医療情報

### 探すタブ
- **検索**: 都市名・英語名・見どころ・旅行スタイルの横断検索
- **天気ストリップ**: 5都市のリアルタイム気温（Open-Meteo API）
- **旅行スタイルガイド**: 6スタイル別の詳細ページ（概要/Tips/規制/推奨都市）
- **おすすめ都市**: 8都市のカード表示
- **都市詳細**: Wikivoyage記事（IndexedDBキャッシュ）+ YouTube動画embed + 見どころチップ

### 旅程タブ
- **旅程作成**: 都市選択 + 泊数 → IndexedDBに保存
- **テンプレート**: 北京3泊4日、上海2泊3日、成都2泊3日、西安2泊3日
- **旅程詳細/編集**: 日程別アクティビティの追加・編集・削除
- **AI旅程プランナー**: プロフィールからClaude AIへの自動プロンプト生成

### レートタブ
- **為替計算機**: Frankfurter API（ECBレート、1時間キャッシュ）
- **8通貨対応**: JPY, USD, EUR, GBP, KRW, AUD, SGD, HKD → CNY変換

### ツールタブ
- VPNガイド、WeChat設定、Alipay設定、緊急連絡先 → 内部ページ遷移
- eSIMガイド、オフラインマップ、翻訳、現金引き出し → 外部サービスリンク

### プロフィール/設定
- プロフィール表示（国籍、ビザ状況、出発日、スタイル、興味）
- 編集（言語/国籍/目的/空港/出発日の変更）
- キャッシュクリア

### PWA / オフライン
- Service Worker（Workbox）によるオフラインキャッシュ
- Wikivoyage記事のIndexedDBプリキャッシュ（人気5都市）
- オフラインバナー表示
- ホーム画面追加でネイティブアプリライクな体験

## テスト手順

### 基本フロー
1. https://china-travel-guide-ashen.vercel.app にアクセス
2. オンボーディング7ステップを完了（日本語 → 日本 → 観光 → 高速鉄道 → 上海浦東 → 文化/グルメ → 完了）
3. 「準備リストを見る」→ ダッシュボードが表示される
4. 各タブ（準備/探す/旅程/レート/ツール）を順にタップ

### 確認ポイント
- [ ] オンボーディング全ステップが正常に遷移する
- [ ] ダッシュボードにカウントダウン・為替レート・チェックリストが表示される
- [ ] チェックリストのチェック状態がリロード後も保持される
- [ ] 探すタブで天気が表示される（API接続時）
- [ ] 都市カードタップ → 詳細ページにWikivoyage記事が表示される
- [ ] 旅程タブでテンプレートから旅程を作成できる
- [ ] 作成した旅程をタップ → 詳細表示・編集ができる
- [ ] レートタブで為替計算ができる
- [ ] プロフィールアイコン（タブバー右上）→ プロフィール表示
- [ ] 編集 → 言語変更 → UIが切り替わる
- [ ] GFWガイドのチェックリストがリロード後も保持される
- [ ] 存在しないURLにアクセス → 404ページが表示される
- [ ] オフライン時にバナーが表示され、キャッシュデータで動作する

## ファイル構成

```
src/
├── api/           # Fetchラッパー + API型定義
├── components/    # 共通コンポーネント（Icon, BottomTabBar, CountryBadge等）
├── data/          # 静的データ（ビザルール, 都市, 緊急連絡先, ガイド等）
├── hooks/         # カスタムフック（useOnlineStatus）
├── pages/         # ページコンポーネント（5タブ + オンボーディング + プロフィール）
├── services/      # APIサービス（rates, weather, wikivoyage, notification, db）
└── store/         # localStorage永続化（userStore）
```

## 制約事項

- **ゼロコスト**: 有料API・サーバーは使用していません
- **クライアントサイドのみ**: すべての処理はブラウザ内で完結
- **ビザ情報**: 2024年末時点のポリシーに基づく。最新情報は大使館で確認してください
- **Wikivoyage記事**: 英語版のみ（API制約）
- **YouTube動画**: キュレーション済みIDのembedのみ（API不使用）
